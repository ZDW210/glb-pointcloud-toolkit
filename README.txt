GLB Point Cloud Toolkit

Files in this folder:

1. GLBPointCloudConverter.exe
   Use this tool to convert a .glb model into:
   - .points.meta.json
   - .points.bin

2. pointcloud-test.html
   Open this file in a browser and load the generated two files for testing.

How to use:

1. Double-click GLBPointCloudConverter.exe
2. Choose a GLB file
3. Set the point count
4. Choose an output directory
5. Click Generate
6. Open pointcloud-test.html
7. Select the generated .points.meta.json and .points.bin files

Suggested point counts:

- Mobile: 80000 to 160000
- Desktop: 200000 to 320000
